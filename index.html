References for HTML & CSS

Tags Name Description
<a> Anchor used to create hyperlinks in an HTML document.
<head> Head Contains general information about an HTML page that isn’t displayed on the page itself.
<title> Title Contains a text that defines the title of an HTML document. The title is displayed in the browser’s title bar
<ol> Orangaized List Ordered list element creates a list of items in sequential order & is numbered.
<ul> Unorganized List Ordered list element creates a list of items in sequential order Uses bullet points rather than numbers.
